# Covid Slayer API

API for Covid Slayer app.

## Install instructions

Copy this repo by:

### `git clone https://github.com/vinieshwan/covid-slayer-server`

Install dependencies by:\

### `npm i`

To run tests:\

### `npm test`

Run by:\

### `npm start`

View the live app at:

### `https://covid-slayer-app.herokuapp.com/`
